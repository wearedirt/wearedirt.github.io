SECTION 2: Bluetooth Setup

Turning on your computer's Bluetooth.

Before you can participate in any of the research studies, please ensure your computer's Bluetooth is turned on.

Windows 11

Press the ⊞ Windows key > Type and select "Bluetooth and other devices settings" > Ensure the toggle is "On".

Connecting the DIRT watch to your computer using Bluetooth

⚠️ Warning: You only need to do this ONCE. Please ensure your DIRT watch is fully charged before going through this step. If you have already done this before, please skip this process.

Windows 11

  1. Press the ⊞ Windows key > Type and select "Bluetooth and other devices settings"

  2. Select "Add device"

  3. Select the first option to add Bluetooth devices

  4. Select "Show all devices"

  5. Select the device that reads "Verisense-01-xxxxxxxxXXXX" where the bolded XXXX matches the ID on your DIRT watch. In this example, the ID is E474.

  6. Enter in "123456" as your pin and press "Connect".

  7. You have now connected your DIRT watch to your computer's Bluetooth devices! Press "Done" to finish.
