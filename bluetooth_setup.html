Pairing the DIRT watch to your computer using Bluetooth

‚ö†Ô∏è Warning: You only need to do this ONCE for every new DIRT watch you receive. Please ensure your DIRT watch is charged before going through this step. If you have already done this before, please skip this process.

üí° Note: There are steps below for Windows 10 and Windows 11 separately. Please select the appropriate section based on your system setup.

Windows 10

  1. Open Bluetooth settings.
  2. Select "Add Bluetooth or other devices".
  3. Select the first option to add Bluetooth devices.
  4. Select "Verisense-01-xxxxxxxxXXXX" where XXXX matches your participant ID. In this example, the ID is E474.
  5. Enter pin "123456" and connect.
  6. Press "Done" when complete.

Windows 11

  1. Open Bluetooth settings.
  2. Select "Add device".
  3. Select the first option to add Bluetooth devices.
  4. Select "Show all devices".
    5. ‚ö†Ô∏è Note: For some Windows 11 machines, you will NOT have the option to "Show all devices". If this is the case, please go to the Windows 11 Alternative instructions below to add your DIRT watch.
  5. Select "Verisense-01-xxxxxxxxXXXX" where XXXX matches your watch. In this example, the ID is E474.
  6. Enter pin "123456" and connect.
  7. Press "Done" when complete.

Windows 11 Alternative

‚ö†Ô∏è Note: This is an alternative way of adding your DIRT watch to your Windows device. If you still have problems adding your DIRT watch using these instructions after trying the instructions above, contact us immediately.
  1. Open Bluetooth settings.
  2. Scroll down until you see the section "Device settings".
  3. In the section, you should see "Bluetooth devices discovery". Please select "Advanced"
  4. After you have selected "Advanced", please scroll back up and select "Add device".
  5. Select the first option to add Bluetooth devices.
  6. Select "Verisense-01-xxxxxxxxXXXX" where XXXX matches your watch. In this example, the ID is E474.
  7. Enter pin "123456" and connect.
  8. Press "Done" when complete.
